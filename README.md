# Inflatable Rental Backend

Backend API for an inflatable rental business with manual admin confirmation.

## Features
- Public inflatables API
- PostgreSQL database with Flyway migrations
- Spring Security (Basic Auth for admin)
- Clean layered architecture (controller / service / repository)

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL
- Flyway
- Maven

## Status
MVP in progress.  
Next steps: booking requests, admin confirmation, availability check.
